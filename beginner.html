Beginner Python

1. Variables and Data Types

Variables store data and can be changed throughout the program. Python has several basic data types including strings, integers, floats, and booleans.


name = "John"           # String
age = 25               # Integer
height = 1.75          # Float
is_student = True      # Boolean

print(f"Name: {name}, Age: {age}")

2. Basic Operators

Operators perform operations on variables and values. Python supports arithmetic, comparison, and logical operators.


# Arithmetic operators
sum = 10 + 5
difference = 10 - 5
product = 10 * 5
quotient = 10 / 5

# Comparison operators
is_equal = (10 == 5)
is_greater = (10 > 5)

3. String Operations

Strings can be manipulated using various methods. Python provides many built-in string operations.


text = "Hello, Python!"
print(text.upper())        # Convert to uppercase
print(text.lower())        # Convert to lowercase
print(text.split(','))     # Split string into list
print(len(text))          # Get string length

4. Lists and List Operations

Lists are ordered collections that can store multiple items. They are mutable and can contain different data types.


fruits = ['apple', 'banana', 'orange']
fruits.append('grape')     # Add item
fruits.remove('apple')     # Remove item
print(fruits[0])          # Access first item
print(len(fruits))        # List length

5. If-Else Statements

Conditional statements control program flow. If-else statements execute different code blocks based on conditions.


age = 18
if age >= 18:
    print("You are an adult")
elif age >= 13:
    print("You are a teenager")
else:
    print("You are a child")

6. For Loops

For loops iterate over sequences. They're commonly used to repeat actions a specific number of times.


# Loop through a range
for i in range(5):
    print(i)

# Loop through a list
fruits = ['apple', 'banana', 'orange']
for fruit in fruits:
    print(fruit)

7. While Loops

While loops repeat as long as a condition is true. They're useful when you don't know the number of iterations in advance.


count = 0
while count < 5:
    print(count)
    count += 1

# Break statement example
while True:
    if count >= 10:
        break

8. Basic Functions

Functions are reusable blocks of code. They help organize code and avoid repetition.


def greet(name):
    return f"Hello, {name}!"

def add_numbers(a, b):
    return a + b

print(greet("John"))
print(add_numbers(5, 3))

9. Dictionaries

Dictionaries store key-value pairs. They provide fast lookups and are highly versatile.


person = {
    'name': 'John',
    'age': 25,
    'city': 'New York'
}
print(person['name'])
person['email'] = 'john@example.com'

10. Input and Output

Python can interact with users through input and output. The input() function reads user input, while print() displays output.


name = input("Enter your name: ")
age = int(input("Enter your age: "))
print(f"Hello {name}, you are {age} years old")

11. Basic Error Handling

Try-except blocks handle errors gracefully. They prevent programs from crashing when errors occur.


try:
    number = int(input("Enter a number: "))
    result = 10 / number
    print(result)
except ValueError:
    print("Please enter a valid number")
except ZeroDivisionError:
    print("Cannot divide by zero")

12. Simple File Operations

Python can read and write files. File operations are essential for data persistence.


# Writing to a file
with open('example.txt', 'w') as file:
    file.write('Hello, World!')

# Reading from a file
with open('example.txt', 'r') as file:
    content = file.read()
    print(content)

13. Basic Math Operations

Python's math module provides additional mathematical functions. These are useful for calculations.


import math

print(math.sqrt(16))      # Square root
print(math.pi)            # Pi value
print(math.ceil(4.3))     # Round up
print(math.floor(4.7))    # Round down

14. Type Conversion

Python can convert between different data types. Type conversion is important for data processing.


# Type conversion examples
num_str = "123"
num_int = int(num_str)    # String to integer
num_float = float(num_str) # String to float
str_num = str(num_int)    # Integer to string

15. Basic String Formatting

String formatting makes text output more readable. Python offers multiple ways to format strings.


name = "John"
age = 25

# Different formatting methods
print("Name: %s, Age: %d" % (name, age))
print("Name: {}, Age: {}".format(name, age))
print(f"Name: {name}, Age: {age}")
